# 🌍 Life Expectancy Analysis – Data Science + Power BI Dashboard  

This project explores global life expectancy data, combining **machine learning** and **interactive dashboards** to uncover insights.  

## 🔹 Project Overview  
- **Goal:** Analyze factors affecting life expectancy across countries and predict outcomes using machine learning.  
- **Approach:**  
  1. Data cleaning & preprocessing (Python)  
  2. Predictive modeling with **XGBoost**  
  3. Explainability using **SHAP** values  
  4. Visualization & storytelling with **Power BI**  


---

## 📊 Power BI Dashboard Preview  


Interactive dashboard includes:  
- KPI cards (Life Expectancy, GDP, Population)  
- Trend analysis (Year-wise global changes)  
- Country comparisons  
- Correlation insights with health & socio-economic factors  

---

## 🛠 Tools & Skills Used  
- **Python:** Pandas, NumPy, XGBoost, SHAP  
- **Power BI:** Data modeling, dashboarding  
- **Data Analytics:** Cleaning, visualization, storytelling  

---

## 📌 Key Insights  
- Socio-economic factors (GDP, Education, Healthcare expenditure) strongly influence life expectancy.   
- SHAP analysis highlighted Income,HIV/AIDS,Adult Mortality driving predictions.  
- Dashboard makes exploration easy for non-technical users.
